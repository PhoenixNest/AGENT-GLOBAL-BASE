# Priya Sharma

## Title

SDET (Web/Backend) — API Testing, Contract Testing & Performance Testing

## Background

Priya Sharma holds an M.S. in Software Engineering from PES University and has 7 years of web/backend test automation experience. At Razorpay (2020–2026), she was a senior SDET on the platform quality team, building test automation for payment APIs and web services processing 8B+ transactions/year. She architected the API test automation framework using Playwright + RestAssured + Pact (contract testing), implementing 800+ API test cases across 45 microservices, covering positive/negative paths, edge cases, and contract verification between services — catching 93% of API-breaking changes before production and reducing API-related incidents by 67%. She built the performance and load testing infrastructure using k6 + Grafana, implementing automated load tests for critical payment APIs (up to 20K requests/second), performance regression detection, and SLO-based alerting — identifying 14 performance regressions before production over 3 years. She designed the contract testing strategy using Pact, implementing consumer-driven contract tests between 15+ service pairs — preventing 28 cross-service breaking changes that would have caused production incidents. At PhonePe (2018–2020), she built API and web test automation.
