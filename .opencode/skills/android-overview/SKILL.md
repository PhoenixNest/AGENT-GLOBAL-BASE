# Android

## Overview

Android platform engineering using modern Kotlin-based tooling. Owned by the Android Development Lead (Kofi Asante-Mensah). All Android code follows MVVM + Clean Architecture with Jetpack Compose for UI, Hilt for DI, Room for persistence, and Retrofit for networking.
