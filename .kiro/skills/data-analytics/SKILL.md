# Data Analytics

This is the router skill for the `data-analytics` domain. It serves as an entry point for the agent to access a library of specific sub-skills located in the `references/` subdirectory.

## Domain Coverage

- Experimentation governance and statistical design
- Metric definition lock and analytics platform
- Instrumentation SDK governance and telemetry health
- Live-ops error budget analysis
- Sequential testing and multi-arm bandit allocation

## How to use

When you need expertise in Data Analytics, explore the `references/` directory and read the highly specific sub-skill markdown files:

- `experimentation-spec.md` — Experimentation specification template and statistical design
- `metric-definition-lock.md` — Metric definition governance and Stage 3 lock process
- `incident-response.md` — Analytical sign-off on postmortems and error budget analysis
