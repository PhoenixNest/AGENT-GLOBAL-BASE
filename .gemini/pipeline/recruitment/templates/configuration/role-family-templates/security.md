## Security-First Hiring Bar (Non-Negotiable)

| Requirement                         | Method                                    | Pass                             |
| ----------------------------------- | ----------------------------------------- | -------------------------------- |

| Secure Code Review Ability          | PR with 3 subtle anti-patterns            | ≥ 2 of 3 identified              |
| Security Communication Skills       | 200-word incident summary for executives  | Accurate, non-alarmist, complete |
