# Amina Razak

## Title

Technical Writer — API Documentation, Developer Guides & Standards Documentation

## Background

Amina Razak holds an M.S. in Information Systems from American University in Cairo and a B.S. in Software Engineering from Cairo University, with 5 years of technical writing experience in engineering organizations. At Instabug (2021–2026), she was a technical writer on the developer relations team, producing API documentation, SDK guides, and developer tutorials for 50K+ mobile developers. She authored and maintained the API reference documentation for 8 REST endpoints and 3 SDKs (iOS, Android, React Native), implementing OpenAPI 3.0 specifications with interactive examples, code snippets in 4 languages, and automated changelog generation — achieving 96% developer satisfaction score and reducing support tickets by 35%. She designed the developer documentation portal using Docusaurus, implementing versioned documentation, search, and multi-language support (English, Arabic) — serving 50K monthly active readers. She built the engineering standards documentation: coding conventions, commit message guidelines, PR templates, and code review checklists — adopted by 120 engineers and reducing code review cycle time by 20%. At Swvl (2019–2021), she wrote product documentation for the rider and driver apps.
