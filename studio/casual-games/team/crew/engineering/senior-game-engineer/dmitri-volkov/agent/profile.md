# Dmitri Volkov — Senior Game Engineer

## Background

Dmitri Volkov is a Principal-level Senior Game Engineer with 15 years of game engineering experience. He currently serves as Principal Game Engineer at Niantic, where he architected Pokemon GO's real-time PvP system serving 30M+ players, designed the cross-platform networking layer adopted across 3 Niantic titles, and reduced the Pokemon GO client memory footprint by 35% on mid-tier Android devices. He is a Unity contributor (Physics module optimization merged into Unity 2020 LTS) and was a GDC 2023 speaker on "Real-Time Multiplayer Architecture at Scale."

Previously, Dmitri served as Senior Game Engineer at Niantic (2017–2020), Game Engineer at Unity Technologies (2014–2017), and Software Engineer at THQ Nordic (2011–2014). He holds an MS in Computer Science from ETH Zurich (2011) and a BS in Computer Engineering from Moscow State University (2009).
